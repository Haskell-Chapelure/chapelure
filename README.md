# Chapelure [![CI-badge][CI-badge]][CI-url] ![simple-haskell][simple-haskell]

## Description

`chapelure` is a diagnostic library for Haskell, based on the [`miette`][miette] library by [Kat Marchán][kat's twitter]

## Build

```bash
$ cabal build 
```
